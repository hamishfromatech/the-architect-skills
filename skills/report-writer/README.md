# Report Writer

Create professional business reports with proper structure and formatting.

## Overview

The Report Writer skill generates various types of business reports including status reports, progress reports, and executive summaries.

## Quick Start

### Trigger Phrases

Use these phrases to activate the skill:
- "write a report"
- "create a status report"
- "generate an executive summary"
- "create a progress report"
- "write a business report"
- "generate a summary report"

## Capabilities

- Generate status reports, progress reports, and executive summaries
- Create analysis reports and review documents
- Apply consistent formatting and structure
- Support multiple output formats

## Report Types

| Type | Purpose | Key Sections |
|------|---------|--------------|
| Status Report | Project status updates | Summary, progress, blockers, next steps |
| Progress Report | Track milestones | Completed, in progress, upcoming |
| Executive Summary | High-level overview | Key points, recommendations |
| Analysis Report | Deep dive analysis | Findings, data, conclusions |

## Report Structure

```markdown
# [Report Title]

**Date**: [Date]
**Author**: [Name]
**Audience**: [Stakeholders]

## Executive Summary
[Brief overview of key findings]

## Key Findings
- [Finding 1]
- [Finding 2]

## Details
[Detailed content]

## Recommendations
1. [Recommendation 1]
2. [Recommendation 2]

## Next Steps
- [Action item 1]
- [Action item 2]
```
